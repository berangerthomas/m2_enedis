src.components Package
======================

The `src.components` package is a key part of the GreenTech Solutions application, responsible for the layout and navigation elements of the user interface. Here, you will find the components that define the **footer** and **navbar** of our application, which ensure a consistent and user-friendly experience.

Submodules
----------

The following modules are included within the `src.components` package:

### src.components.footer module
The `footer` module provides the footer component for the application, including any links, copyright notices, and additional navigation elements displayed at the bottom of each page.
