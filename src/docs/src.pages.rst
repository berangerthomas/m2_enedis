Application Pages Overview
==========================

Below is an overview of the main pages in our application, each serving a unique purpose:

- **Charts Page**: Displays various graphs for analyzing key metrics.
- **Context Page**: Provides background information on the application and details about the dataset used.
- **Home Page**: The main interface for prediction, allowing users to input data and receive predictions.
- **Map Page**: Shows a map displaying all residential properties in department 69, with additional information on each location.
- **404 Not Found Page**: Error page shown when a user navigates to a non-existent or unavailable page.
- **Consumption Prediction Page**: Enables users to forecast energy consumption based on input data.
- **DPE Prediction Page**: Offers predictions for the DPE (Diagnostic de Performance Énergétique) rating of properties.
